# ĐÁNH GIÁ LẠI — nơi hệ thống đối diện những lần nó sai

- Tín hiệu đóng → chuyển file từ `tin-hieu/` vào đây, điền đủ phần "KHI ĐÓNG"
- Mỗi quý: tổng kết 1 file `tong-ket-QX-YYYY.md`:
  - Tỷ lệ tín hiệu đúng tại mốc kiểm chứng
  - Độ sớm trung bình so với truyền thông đại chúng
  - Hiệu suất giả định so với VN-Index
  - 3 sửa đổi đề xuất cho hệ thống (nếu có) — đây là CỬA DUY NHẤT mở lại nhánh đã khóa 🔒
