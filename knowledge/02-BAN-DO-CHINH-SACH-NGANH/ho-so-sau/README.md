# HỒ SƠ SÂU — TẦNG 2

Chỉ mở khi nhóm đạt 🟠 trở lên. Mỗi nhóm 🟠 = 1 thư mục con:

```
ho-so-sau/
└── YYYY-MM-nhom-nganh/
    ├── van-ban-goc/          ← lưu/link các văn bản số hiệu đầy đủ
    ├── chuoi-tac-dong.md     ← phân tích chi tiết + độ trễ + định lượng nếu có
    └── cham-diem-ma.md       ← chấm từng mã theo khung 03, đủ 4 lớp
```

Nhóm rời khỏi 🟠/🔴 quá 3 tháng → chuyển thư mục vào `_luu-tru/` (không xóa —
là dữ liệu học cho backtest sau này).
